Angular plugin docs

Angular docs

Angular is a framework for developing typescript apps from Google. In order to integrate progress-up with your angular project please follow the instructions in this document.

First install the plugin for Angular.

	$ npm install progress-up-angular
# or 
	$ yarn add progress-up-angular
  • Create a new project
 $ ng new progress-up-test
 $ cd progress-up-test
 $ ng serve

After you get the scaffolding up you can proceed to work on integrating the plugin into your project.

This is just an example. Your situation may be different.

This is src/app/app.component.html.

<progress-up></progress-up>

This is src/app/app.component.ts.

import { Component, OnInit } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html'
})
export class AppComponent {

}

This is src/app/app.module.ts.

import { BrowserModule } from '@angular/platform-browser';
import { NgModule } from '@angular/core';
import { HttpClientModule } from '@angular/common/http';

import { AppComponent } from './app.component';
import { ProgressUpModule } from 'progress-up-angular';

import { FormsModule, ReactiveFormsModule } from '@angular/forms';

@NgModule({
  declarations: [
    AppComponent
  ],
  imports: [
    BrowserModule,
    HttpClientModule,
    ProgressUpModule,
    FormsModule,
    ReactiveFormsModule
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }
